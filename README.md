# Tools Sharing Website

A web-based platform for lending and borrowing tools within a community.

## About
This is the web version of the Tools Sharing App. Users can list their tools, browse available tools, and send requests to borrow them — all through a browser.

## Features
- User authentication (login and signup)
- List tools available for lending
- Browse and request tools from other users
- Availability management
- Easy and simple interface

## Tech Stack
- HTML
- CSS
- JavaScript
